# Shopify Theme Copilot

AI-powered Shopify Theme Performance Copilot.

## Features

- Lighthouse Audit
- Theme Performance Analysis
- CSS/JS Optimization Suggestions
- AI-generated Performance Report (Coming Soon)

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Commands

```bash
pnpm dev audit
```
